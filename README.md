## 📘 Descrição do Projeto
Projeto que busca entregar uma prévia de internação dos pacientes mediante validação de fatores que independem do médico.
Desta forma o internado e a sua família conseguiriam estimar os custos quem podem ter durante toda a internação.

 
## 🛠 Tecnologias Utilizadas
- Linguagem: Python
- Ferramentas: scikit-learn, scipy, joblib, matplotlib, seaborn, pandas, numpy
